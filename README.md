# Black History: Leaders & Innovators

אתר חינוכי בעברית על אנשים שחורים בעלי השפעה בהיסטוריה, בשני תחומים מרכזיים:

1. מנהיגים ופעילי זכויות אדם
2. מדע, המצאות וחדשנות

האתר נבנה כפרויקט סטטי פשוט המתאים ל-GitHub Pages.

## מטרות האתר

- להציג ביוגרפיות קצרות וברורות של דמויות מרכזיות.
- להסביר את ההשפעה ההיסטורית של כל דמות.
- לחבר בין מאבקי חירות, זכויות אדם, מדע וחדשנות.
- לשמש בסיס עתידי לאנציקלופדיה מורחבת עם עמוד נפרד לכל אישיות.
